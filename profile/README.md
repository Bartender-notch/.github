## What is this Bartender Menu Bar Integration?
**Bartender** is the essential macOS utility for anyone who has too many menu bar icons — which is almost everyone with a modern MacBook (especially those with a notch) or anyone who uses multiple apps. This repository serves as a centralized resource hub for Mac users, designers, developers, writers, and productivity enthusiasts seeking to master **menu bar decluttering**, **notch management**, **icon rearrangement**, **hidden icons** (show on click), **triggers** (show icons when certain conditions are met), **Bartender bar** (secondary menu bar), **search**, **keyboard shortcuts**, and **automatic organization**. Bartender solves a simple but critical problem: the macOS menu bar has limited space, especially on 14″ and 16″ MacBook Pros with a notch (which eats into the menu bar area). Bartender lets you hide icons you don't need constant access to, while keeping them one click away. Whether you are a creative with 20+ menu bar apps, a developer with tools like Docker and iStat Menus, or anyone who wants a clean, focused menu bar, understanding the **Bartender menu bar pipeline** is essential for reclaiming screen real estate and reducing visual clutter.

## Core Features & Capabilities
- **Menu Bar Decluttering:** Hide any menu bar icon. Move icons between Bartender bar (secondary bar) and main menu bar. Show hidden icons on click (Bartender bar appears). Icons remain accessible but not always visible. Supports all menu bar apps (system icons, third‑party apps).
- **Notch Management:** Critical for MacBooks with notches (2021+). The notch physically cuts into the menu bar area, reducing available space for icons. Bartender hides icons behind the notch, shows them in Bartender bar. Prevents icons from being obscured by the notch. Works with any notch MacBook.
- **Icon Rearrangement:** Reorder menu bar icons (drag and drop). Change order of both visible and hidden icons. Group related icons together (e.g., all cloud apps, all audio apps). Separate from macOS's limited reordering.
- **Bartender Bar (Secondary Bar):** A secondary menu bar that appears when you click the Bartender icon (or keyboard shortcut). Hidden icons live in the Bartender bar. Bartender bar can be displayed as: popup (click to show), always visible (like second row), or on hover. Fully customizable (position, style, animation).
- **Triggers (Conditional Visibility):** Show certain icons only when conditions are met. Examples: "Show VPN icon only when VPN is connected", "Show battery icon only when on battery", "Show Time Machine icon only when backing up", "Show Do Not Disturb icon only when enabled". Triggers keep menu bar clean while ensuring you see icons when they matter.
- **Search & Keyboard Navigation:** Search for any menu bar icon (Cmd+Shift+B). Type app name, Bartender highlights icon. Keyboard navigation (arrow keys, Enter to click). Great for when you have 30+ icons and can't remember where something is.
- **Hotkeys:** Assign keyboard shortcuts to show Bartender bar. Show/hide specific icons via shortcut. Trigger search. Control Bartender without mouse.
- **Automatic Organization:** Rules to automatically move new icons to Bartender bar (or keep in main bar). Set default behavior for all new apps. Never manually move a new icon again.
- **Hidden Icons in Menu Bar:** Some apps have hidden menu bar icons (e.g., Dropbox's "hidden" icon). Bartender can show these hidden icons. Useful for apps that try to be "minimal" but you want them visible.
- **Updates & New Apps:** Monitors for new menu bar icons. When a new app appears, Bartender can auto‑hide it or ask what to do. Prevents clutter from newly installed apps.
- **Export/Import Settings:** Save your Bartender configuration (which icons are hidden, order, triggers). Share with colleagues. Restore after macOS reinstall. Sync between Macs via Dropbox or iCloud.
- **Dark Mode & Customization:** Adapts to macOS Dark Mode. Customize Bartender bar appearance (color, opacity, position). Choose which side of menu bar Bartender icon appears. Adjust animation speed.

## Bartender Optimization Techniques
To ensure a clean and productive menu bar workflow, use **Triggers** to show icons only when needed — VPN icon only when connected, battery icon only when on battery. The **Bartender menu bar workflow** benefits from **Notch Management** — hide icons behind the notch, access via Bartender bar. Use **Icon Rearrangement** to put your most‑used icons in the main menu bar (time, battery, Wi‑Fi, sound), less‑used icons in Bartender bar. Use **Hotkeys** to show Bartender bar without clicking (Cmd+Shift+B). Use **Search** when you have dozens of icons and can't find the one you need. Use **Auto‑Hide Rules** for new apps — Bartender automatically hides new icons, you can promote them if needed.

## Supported Integrations
- **macOS:** Native integration with macOS menu bar, notch MacBooks, Dark Mode, Accessibility APIs.
- **All Menu Bar Apps:** Works with every app that puts an icon in the menu bar (system apps, third‑party apps, background apps). No app‑specific integration needed.
- **Third‑Party:** Dropbox, Google Drive, iStat Menus, 1Password, Alfred, BetterTouchTool, Magnet, Tidal, Spotify, Discord, Slack, Zoom, Docker, Parallels, VMware, and thousands more.
